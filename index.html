Sneer Your Own Social Network

Sneer is an open source, fine-grained component platform for P2P social network applications. It is a pojo-style crossover between Skype and OSGi on a serious XML-free diet. It completely dismisses questionable middlemen such as email providers, Google, Facebook, etc; and sets the scene for applications that empower users to control and share their information and computing resources with their freinds in an autonomous, sovereign manner. Users are able to create, share and assemble their own tiny components called bricks (as in Lego) to build Sovereign Applications (Snapps).

Install

Click below to download Sneer source code, documentation, binaries and run them using Java 6 Webstart. You can access the documentation from within the Sneer main menu.
